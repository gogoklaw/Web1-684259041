แบบฝึกปฏิบัติ : HTML Formatting

เรียนรู้การจัดรูปแบบข้อความด้วยแท็ก HTML พร้อมตกแต่งด้วย CSS

📄ตัวอย่างผลลัพธ์

B: ข้อความตัวหนา

STRONG: ข้อความสำคัญ

I: ข้อความเอียง

EM: ข้อความเน้นย้ำ

U: ข้อความขีดเส้นใต้

MARK: ข้อความที่ถูกเน้น

SMALL: ข้อความขนาดเล็ก

DEL: ข้อความที่ถูกลบ

SUP: x2

SUB: H2O

</> ตัวอย่างโค้ด HTML

1<b>ข้อความตัวหนา</b>
2<strong>ข้อความสำคัญ</strong>
3<i>ข้อความเอียง</i>
4<em>ข้อความเน้นย้ำ</em>
5<u>ข้อความขีดเส้นใต้</u>
6<mark>ข้อความที่ถูกเน้น</mark>
7<small>ข้อความขนาดเล็ก</small>
8<del>ข้อความที่ถูกลบ</del>
9<sup>2</sup>
10H<sub>2</sub>O

🎯กิจกรรมฝึกปฏิบัติ

✓ ให้นักศึกษาสร้างหน้าเว็บ 1 หน้า

เกร็ดความรู้

แท็ก HTML Formatting แต่ละแท็กมีหน้าที่แตกต่างกัน เช่น <strong> ใช้เน้นความสำคัญของข้อความ <em> ใช้เน้นข้อความ <mark> ใช้ไฮไลต์ข้อความ และ <del> ใช้แสดงข้อความที่ถูกลบ การเลือกใช้แท็กอย่างถูกต้องจะช่วยให้เว็บไซต์มีมาตรฐานและอ่านง่ายมากขึ้น
