แบบฝึกปฏิบัติ: HTML Heading

ฝึกใช้แท็ก <h1> – <h6> และการปรับขนาดหัวข้อด้วย CSS

  • H1:  

    หัวข้อหลักของหน้าเว็บ

  • H2:  

    หัวข้อรอง

  • H3:  

    หัวข้อย่อยระดับ 3

  • H4:  

    หัวข้อย่อยระดับ 4

  • H5:  
    หัวข้อย่อยระดับ 5
  • H6:  
    หัวข้อย่อยระดับ 6

ตัวอย่างการใช้


            <H1> Welcome to My Website </H1>
            <H2> HTML Basics </H2>
            <H3> What is HTML? </H3>
            <H4> HTML Syntax </H4>
            <H5> Example Code </H5>
            <H6> Note: Case Sensitive</H6>

เมื่อแสดงผลจะได้ขนาดลดหลั่นแบบนี้:



                    
H1: Welcome to My Website

                    
H2: HTML Basics

                    
H3: What is HTML?

                    
H4: HTML Syntax

                    
H5: Example Code
 
                    
H6: Note about heading size
